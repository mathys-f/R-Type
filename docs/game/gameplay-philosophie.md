# Gameplay philosophie

The game will be created for multiplayer experience, the enemies in the game will be bigger with lots of hp (healthpoint) to encourage playing together.

Only 5 button will be used :

- Up arrow : Make your cruiser go up.
- Right arrow : Make your cruiser go right.
- Left arrow : Make your cruiser go left.
- Down arrow : Make your cruiser go down.
- Space : Make your cruiser shoot.

And if controller are implemented :

- Joystick : Move the cruiser in any direction.
- Trigger : Make your cruiser shoot.

These simple configuration allow player to play with a single hand.
