# Game Design

The game will start with the cruiser(s) appearing at the left, when the cruiser is at the 3/4 of the screen tha player(s) will take control and the screen will begin scrolling. Enemies will start appearing on the screen that the player need to dodge or shoot down, this will go on for a certain time or distance and then the boss will appear, the player(s) need to defeat him to win the game.
